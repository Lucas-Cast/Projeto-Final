Um site da copa do mundo que seja capaz de informar as pessoas sobre coisas como a data dos jogos, seleções e jogadores participantes, placares e classificação. Também tenha uma área de cadastro de usuários conectadas com um banco de dados, onde o usuário defina um nome e uma senha, para que este possa ter um acesso mais amplo do site. Deverá haver uma aba com links para  redirecionar os usuários para sites instituições de caridades relacionadas ao esporte. O usuário será capaz de interagir a partir de uma seção de comentários, para que possa avaliar e dissertar sobre as seleções, os jogos ou até mesmo os jogadores, tais comentários e avaliações precisam ser armazenados no banco de dados de modo que possam ser lidos por outros usuários. Além disso, deve haver uma moeda fictícia atribuída à quem fizesse algumas tarefas dentro do site, esta poderá ser usada em uma loja interna que venderá cosméticos para serem colocados no perfil de quem os comprar.
